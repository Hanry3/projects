# Login Form

This is a simple login page created using **HTML**, **CSS**, and **JavaScript**. The page is fully responsive and includes basic validation for the user inputs.

## Features

- **Responsive Design**: The layout adjusts for different screen sizes, ensuring a consistent user experience on mobile, tablet, and desktop.
- **Input Validation**: The form ensures that all fields are filled in correctly, checking the format of the email and enforcing password requirements.
- **Smooth Animations**: Animations provide a better user experience, with transitions for input focus and hover effects.

## Technologies Used

- **HTML5**: For structuring the login page content.
- **CSS3**: For styling, including animations and responsive design.
- **JavaScript**: For form validation and interactivity, such as the password visibility toggle.
